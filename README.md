# Horror-Game-in-Unreal-Engine
 "Five Restante at FMI" it's an horror-survival game, made in Unreal Engine. This repository contains the C++ source code of the game like the RRT algorithm, the RSA encryption and other technologies used.
 
Acest repository conține exclusiv implementările algoritmice C++, decuplate de asset-urile vizuale și de structura arhitecturală motorului Unreal Engine.
